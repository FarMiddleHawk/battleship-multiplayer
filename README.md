# Battleship Multiplayer

Real-time 2-player Battleship game built with Node.js, Express, and Socket.io.

## Deploy to Render

1. Push this repo to GitHub
2. Go to https://render.com
3. Create new Web Service
4. Connect your GitHub repo
5. Set build command: `npm install`
6. Set start command: `npm start`
7. Deploy!

Your game link will be provided. Share it with your opponent to join!

## Local Development

```bash
npm install
npm start
```

Open http://localhost:3000
